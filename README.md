## Usage

> 没有对错，只是觉得不理解

不理解为什么能写这么多字数
不理解这种观点会有这么多人赞同
不理解为什么现状是这样
不理解为什么可以写的这么好
不理解为什么有这么牛逼的人存在

如果你也有不理解，尽管提交PR，开放，随意发表自己的不理解，还有为了保证真实性和可靠行，请附上链接：），thak～
